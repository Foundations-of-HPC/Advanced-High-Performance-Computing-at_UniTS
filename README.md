# Advanced High Performance Computing @ University of Trieste
materials, slides and exercises from the advanced HPC course @ units in years 2024, 2025
---

This repository is maintened by Luca Tornatore from INAF - National Institute for Astrophysics, Italy.
It collects slides, materials, code examples, assignment, from the course in Advanced High Perofrmance Computing since year 2024
(previous editions are hosted in separate repos, still under this git)

The topics covered are (note credits to external collaborators)
- Vectorization
- OpenMP Tasks
- OpenMP threads synchronization
- Intro to threads by using pthreads (M. Poggi, from ExactLab)
- MPI One-sided communications
- MPI shared-memory windows
- MPI Communicators
- GPU offloading with OpenMP, OpenACC and CUDA (G. Taffoni, INAF; N. Shukla, M. Redenti, CINECA)
- MPI and GPU with Python (G. Karadermi, LMU)
